# Mercado Phovl 

Este repositorio contiene dos proyectos que trabajan juntos con la misma base de datos en la nube:

- **Mobile (Android)** → carpeta `/mobile`
- **Web** → carpeta `/web`

## Cómo correr cada proyecto

### Mobile
1. Abrir `/mobile` en Android Studio.
2. Configurar dependencias con Gradle.
3. Ejecutar en emulador o dispositivo.

### Web
1. Entrar a `/web`.
2. Instalar dependencias: `npm install`.
3. Ejecutar: `npm start`.
